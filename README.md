# 프로젝트명: 칼로리를 부탁해

📑 1. 개요 <br>
최근에 다이어트에 대한 관심이 많아지면서 다이어트 보조제에 수요도 늘어나
다이어트 보조제 사이트를 Oracle DB와 JSP를 활용하여 홈페이지 CRUD를 구현하여 제작하게 되었습니다

🔧 2. 프로젝트 개발환경 <br>
A. java 1.8 (Jdk. 1.8) <br>
B. Oracle 11g <br>
C. apache-tomcat-8.5.78 <br>
D. eclipse 2021-06

🛠️ 3-1. 구현 기능 <br>
